Como solucionar problemas de las aplicaciones de Microsoft Store



Si su ordenador tiene Windows 10 y tiene problemas con alguna aplicación de Microsoft Store, debe tener en cuenta estas correcciones.Primero inicie sesión con su cuenta de Microsoft y en el siguiente orden:

1. Verifique que Windows tenga la actualización mas reciente: en el cuadro de búsqueda escribe buscar actualizaciones y haz clic. En caso de que encuentre una actualización disponible haz clic en Instalar ahora.

2. Verifique que la aplicación funcione con Windows 10.

3. Para actualizar Microsoft Store: en caso de que Microsoft store no este actualizada haz clic en inicio y busca Microsoft store, luego selecciona el icono de descarga que lo enviara a la lista de aplicaciones que necesitan actualización, si Microsoft store se encuentra en esta lista tendrá que actualizar la aplicación.

4. Puedes reparar o reiniciar sus aplicaciones

5. Para reinstalar las aplicaciones busca en Microsoft Store y selecciona Mi biblioteca. Selecciona la aplicación que quiera reinstalar y selecciona Instalar.

6. Por ultimo puedes ejecutar el solucionador de problemas, luego haz clic en inicio➔ configuración➔ actualización y seguridad ➔ solución de problemas➔ aplicaciones de Windows store➔ ejecuta el solucionador de problemas.
